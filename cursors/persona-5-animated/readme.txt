=== PERSONA 5 ANIMATED Cursor Set ===

By: THTH

Download: http://www.rw-designer.com/cursor-set/persona-5-animated

Author's description:

Animated cursor set from the game Persona 5, made on a 128-pixel canvas. Therefore, we need to modify the Windows cursor size to '3' for it to look perfect, or '2' for it to appear with slightly less detail!

ESP

Set de cursores animados del juego persona 5, estan hechos en un lienzo de 128 pixeles, por ende debemos modificar el tamaño del cursor propio de windows al numero "3" para que se vea perfecto o numero 2 para que se vea algo con menos detalle!!!
